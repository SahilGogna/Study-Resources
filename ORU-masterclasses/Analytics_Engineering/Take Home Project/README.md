# NorthPay Analytics Modeling Take-Home

## Overview

This project simulates a real-world FinTech analytics modeling exercise.

You are given raw payment processor data and asked to design a complete analytical layer to support business reporting.

The focus is on:
- Data modeling
- Grain definition
- Financial correctness
- Handling edge cases
- Clear documentation

## What You Receive

- 4 raw CSV datasets
- Business requirements
- Setup instructions
- Evaluation rubric
- Submission templates

## What You Submit

A GitHub repository containing:
- Your modeling SQL
- Your grain decisions
- Your metric definitions
- A professional README

This project is designed to simulate a real analytics engineering take-home.
