# Cybersecurity Engineer Roadmap (Resource Doc)

---

## Why Cybersecurity in Canada Right Now

- Canada has one of the largest cybersecurity talent gaps in the G7
- Over 25,000 cybersecurity positions remain unfilled in Canada annually
- Entry-level roles start at $75,000–$95,000 CAD
- Mid-level roles (3–5 years) average $110,000–$140,000 CAD
- Federal government, banks, and healthcare are the biggest hirers
- Most roles do **not** require a CS degree — certifications carry equal or more weight

---

## Step 1 — Build Your Foundation

### Networking Basics (Non-negotiable)

- Understand TCP/IP, DNS, HTTP/HTTPS, firewalls, VPNs
- Learn subnetting and basic network architecture
- Understand how data moves across a network

### Operating Systems

- Get comfortable with Linux (Ubuntu or Kali)
- Understand Windows Server basics — most Canadian enterprise environments run Windows

### Free Resources

- Professor Messer (free CompTIA study material) — [professormesser.com](http://professormesser.com)
- NetworkChuck on YouTube — excellent for networking fundamentals
- TryHackMe — Pre-Security learning path (free tier available)

---

## Step 2 — Entry-Level Certifications (Start Here)

### CompTIA Security+

- Most recognized entry-level security cert in Canada
- Required or preferred in 60%+ of Canadian entry-level job postings
- Cost: ~$400 CAD
- Study time: 2–3 months
- Free prep: Professor Messer Security+ course

### CompTIA Network+ (Optional but helpful)

- Good if you have zero networking background
- Pairs well with Security+

### Google Cybersecurity Certificate (Coursera)

- Free with financial aid
- Good beginner foundation before attempting Security+
- Recognized by some Canadian employers for entry-level SOC roles

## Step 3 — Hands-On Technical Skills

### Tools You Must Know

- **Wireshark** — network traffic analysis
- **Nmap** — network scanning
- **Metasploit** — penetration testing framework
- **Burp Suite** — web application security testing
- **Splunk** — SIEM tool used heavily in Canadian SOC teams
- **Kali Linux** — penetration testing OS

### Where to Practice

- **TryHackMe** — beginner friendly, guided rooms, free tier available
- **HackTheBox** — more advanced, respected in the community
- **PicoCTF** — beginner CTF challenges
- **DVWA (Damn Vulnerable Web App)** — local practice environment

---

## Step 4 — Choose Your Specialization

Three paths with the highest demand in Canada:

### Path A — SOC Analyst (Security Operations Center)

- Monitor and respond to security incidents
- Most available entry-level roles in Canada
- Key tools: Splunk, Microsoft Sentinel, CrowdStrike
- Target cert: CompTIA Security+ → CompTIA CySA+

### Path B — Cloud Security

- Fastest growing specialization in Canada
- Most Canadian enterprises are on AWS or Azure
- Key skills: IAM, cloud network security, CSPM tools
- Target cert: AWS Security Specialty or Microsoft SC-200

### Path C — Penetration Testing / Ethical Hacking

- Highest paid specialization
- Most competitive to break into
- Requires strong hands-on portfolio
- Target cert: CEH → OSCP

## Step 5 — Advanced Certifications (Canadian Employers Actually Ask For These)

| Certification | Role | Difficulty | Cost (CAD) |
| --- | --- | --- | --- |
| CompTIA CySA+ | SOC Analyst | Intermediate | ~$450 |
| CEH (Certified Ethical Hacker) | Pen Tester | Intermediate | ~$600 |
| OSCP (Offensive Security) | Pen Tester | Advanced | ~$1,700 |
| AWS Security Specialty | Cloud Security | Intermediate | ~$400 |
| Microsoft SC-200 | Cloud/SOC | Intermediate | ~$250 |
| CISSP | Senior/Management | Advanced | ~$800 |

> **Note:** For entry-level, Security+ is enough to start applying. OSCP and CISSP are for 2–3 years into your career.

## Step 6 — Build Your Portfolio

### What to include

- **TryHackMe profile** — share your badge progress publicly
- **HackTheBox profile** — include your rank
- **GitHub writeups** — document CTF solutions and home lab projects
- **Home lab** — set up a virtual network using VirtualBox or VMware with vulnerable VMs

### Portfolio project ideas

- Build a basic SIEM using Splunk free tier and simulate log analysis
- Set up a vulnerable web app and document your pentest findings
- Write a threat analysis report on a publicly disclosed CVE
- Create a network diagram and security audit for a fictional company

## Step 7 — Canadian Companies Actively Hiring

### Financial Sector (Biggest hirers)

- RBC, TD Bank, Scotiabank, BMO, CIBC — all have dedicated cybersecurity teams
- Desjardins (Quebec-based, large security team)

### Government & Public Sector

- Communications Security Establishment (CSE) — Canada's national cybersecurity agency
- Department of National Defence
- Canada Revenue Agency
- Provincial governments (Ontario, BC, Alberta have the largest tech teams)

### Tech & Consulting

- Telus, Bell, Rogers — all have large security operations
- IBM Canada, Accenture, Deloitte, KPMG — big demand for security consultants
- CGI Group — largest Canadian IT company, heavy government contracts
- Herjavec Group — Canadian-founded cybersecurity firm

### Startups & Scale-ups

- Corelight, Arctic Wolf (Canadian HQ), Beauceron Security, eSentire

## Step 8 — Job Search Strategy for Canada

### Where to Apply

- **LinkedIn** — filter by Canada, posted in last 7 days, cybersecurity
- **Government of Canada Job Bank** — [jobbank.gc.ca](http://jobbank.gc.ca) (underused, real postings)
- **Cyber Job Central** — cybersecurity-specific job board
- **Indeed Canada** — search SOC analyst, security analyst, information security

### Keywords to Use in Your Resume

- SOC Analyst, Information Security Analyst, Cybersecurity Analyst
- Threat Detection, Incident Response, Vulnerability Assessment
- SIEM, Firewall, IDS/IPS, Endpoint Security
- CompTIA Security+, CySA+, OSCP (only if you have them)

### Canadian Resume Tips

- One page if under 5 years experience
- Include your certifications prominently near the top
- Quantify everything — "reduced incident response time by 30%" beats "helped with incidents"
- No photo, no date of birth — Canadian standard
